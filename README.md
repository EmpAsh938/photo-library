### Intro
Photo Library is a free stock image app.

### Usage
```
git clone https://github.com/EmpAsh938/photo-library
cd photo-library/client
npm install
npm start
```

### Live Preview
[Visit Photo Library](https://empash-photo-library.netlify.app)
